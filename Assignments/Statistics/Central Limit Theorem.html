Problem 1:¶

Let X be a random variable with μ = 10 and σ = 4. A sample of size 100 is taken from this population. Find the probability that the sample mean of these 100 observations is less than 9.

Explanation:¶

μ(population)  = 10
σ (population) = 4
Sample size N = 100
p(μ_sample < 9) = ?

σ (sampling distribution) = σ (population)/ sqrt(N) = 4/ sqrt(100) = 0.4
z-score = (x-u)/σ s.d = 9-10/0.4 = -1/0.4 = -2.5
p(-2.5) = 0.0062


prob = norm.cdf(-2.5)
print(prob)

Answer: 0.0062¶

def calc_prov(pop_mean,
              standard_deviation,
              sample_size,
              random_value=None,
              mean_value=None,
              prob_random_variable=True,
              prob_for_mean=False):
    """
    Sample programme to 
    i) calculate probabiliy for normal random variable at a given value and
    ii) calculate probaility for at mean value of the sample of size N 
           
    """
    u = pop_mean
    usample = pop_mean
    sd = standard_deviation
    N = sample_size
    sd_sample = standard_deviation/(np.sqrt(N))
    
    if prob_random_variable:
        zrv = (random_value - u) / sd
        p_rv = norm.cdf(zrv)
    if prob_for_mean:
        zpv = (random_value - u) / sd_sample
        p_pv = norm.cdf(zpv)
    
    return p_rv,p_pv
